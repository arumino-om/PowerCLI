# PowerCLI
簡単な構文でコンピューターの簡易的な電源操作を行うCLIツール

## 使い方
以下の構文でコマンドを実行してください。
```
pwcli <command>
```

### コマンド一覧
|動作|コマンド|説明|
|:--|:--|:--|
|シャットダウン|`pwcli shutdown`|コンピューターをシャットダウン|
|再起動|`pwcli reboot`|コンピューターを再起動|
|ロック|`pwcli lock`|コンピューターをロック|
|スリープ|`pwcli sleep`|コンピューターをスリープ|
|スリープ(ウェイクアップなし)|`pwcli sleep-de`|システムウェイクアップイベントを無効にしてスリープ|
|ハイバネート|`pwcli hibernate`|コンピューターをハイバネート|
